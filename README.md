# Antigravity Mobile Monitor Launcher

Mobile web interface for monitoring Antigravity chat.

## Prerequisites

Start Antigravity with remote debugging enabled:

```bash
antigravity . --remote-debugging-port=9000
```

Make sure the Cascade chat panel is open in Antigravity.

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

Then access the monitor at `http://localhost:3000` or from mobile at `http://<your-ip>:3000`.
